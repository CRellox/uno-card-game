# UNO! The Card Game (Unfinished Project) JDK21

A Java implementation of the classic UNO card game. Developed as a personal project to learn Java language, 

experiment with game development, and practice real-time interactions using JDK 21.

>  **Note:** This is an unfinished project focused on learning and experimentation.

# Learning Objectives.

1. Java syntax and best practices
2. Object-oriented design principles
3. Algorithm implementation
4. Console-based user interfaces

# Current Implementation
The project currently includes:

1. Basic card and deck structures
2. Player management system
3. Core game loop foundation
4. Partial rule implementation
